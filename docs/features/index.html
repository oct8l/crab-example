March 19, 2017

Crab also supports tags. This page has some tags, as you can see below, although tags are probably more common with blog posts. If you click on a tag, you’ll be redirected to a page that shows all pages and posts with that tag. (For some tags, there is only this one page.)

Nested menus

March 18, 2017

Crab supports nested menus up to three levels deep. This is especially helpful for websites that focus on content with a fixed, pre-defined strucure.
